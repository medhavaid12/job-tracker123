# 💼 Job Tracker Dashboard

A modern **Job Tracking Web Application** that helps users manage job applications, monitor progress, and stay organized throughout their job search journey.

Built with a clean UI and structured workflow, this app provides a centralized dashboard to track applications, interviews, offers, and more.

---

## ✨ Features

### 🔐 Authentication
- Secure Login & Signup system  
- User session handling  
- Clean and minimal login UI  

### 📊 Dashboard Overview
- View key metrics:
  - Total Jobs  
  - Applied Jobs  
  - Interviews  
  - Offers  
  - Rejected  
- Personalized welcome screen  
- Quick access actions  

### 🧾 Job Management
- Add new job applications  
- View all applications  
- Track job details:
  - Company name  
  - Role (e.g., Developer)  
  - Location  
  - Salary  
  - Job type  

### 📌 Status Tracking
- Track job stages:
  - Applied  
  - Interview  
  - Offer  
  - Rejected  
- Visual indicators for each status  

### ⭐ Additional Modules
- AI Parser  
- Saved Jobs  
- Recommendations  
- Analytics Dashboard  
- Profile Management

## ⚙️ Installation & Setup

### 1️⃣ Clone the repo
git clone https://github.com/medhavaid12/job-tracker123.git  
cd job-tracker123  

### 2️⃣ Install dependencies
npm install  

### 3️⃣ Run the app
npm run dev  

### 4️⃣ Open in browser
http://localhost:5173  

---

## 🔑 Environment Variables (if backend used)

Create a `.env` file:

VITE_API_URL=http://localhost:5000  
MONGO_URI=your_mongodb_uri  
JWT_SECRET=your_secret
